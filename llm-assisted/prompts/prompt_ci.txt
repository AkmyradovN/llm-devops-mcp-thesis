You are a senior DevOps engineer. Generate a complete GitHub Actions workflow YAML file that deploys MCP servers to an EC2 instance.

- Workflow name: "Deploy MCP Servers (LLM-Assisted)"

12. Print deployment summary with total duration in seconds (always runs, even on failure)
13. Clean up: remove the SSH key file (always runs)

IMPORTANT:
- Use StrictHostKeyChecking=no for all SSH/SCP commands
- Use set -e in SSH heredocs so failures propagate
- The health check steps must FAIL the pipeline if the server doesn't respond

Return your response as a single JSON object with exactly this key:
- "deploy_yml": the complete deploy.yml contents as a string (valid YAML)
